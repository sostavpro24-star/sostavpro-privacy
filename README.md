# Политика конфиденциальности

Приложение «Состав.Про» уважает конфиденциальность пользователей.

## Сбор данных
Приложение не собирает, не хранит и не передаёт персональные данные пользователей.

## Камера
Камера используется исключительно для сканирования состава продуктов.
Изображения не сохраняются и не передаются на сторонние серверы.

## Аналитика и сторонние сервисы
Приложение не использует сторонние аналитические сервисы и не передаёт данные третьим лицам.

## Cookies и трекинг
Приложение не использует cookies и не осуществляет трекинг пользователей.

## Контактная информация
По всем вопросам, связанным с конфиденциальностью, вы можете связаться с нами по электронной почте:

## Изменения
Политика конфиденциальности может быть обновлена без предварительного уведомления.
